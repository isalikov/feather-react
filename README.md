-   [Documentation](https://isalikov.github.io/feather-react)

#### Installation

```shell
npm i --save feather-react
```

#### Usage

```tsx
import Icon from 'feather-react'

const MyComponent: React.FC = () => <Icon size={16} name="feather" />
```
